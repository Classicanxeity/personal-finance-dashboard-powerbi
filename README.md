# Personal Finance Dashboard 

## Overview
Advanced Personal Finance Dashboard analyzing 266 transactions 
across 12 months built as part of US CMA journey.

## Tools Used
- Power BI Desktop
- Power Query
- DAX (10 measures)
- Excel / CSV

## Key Features
- Income vs Expense trend analysis
- Cumulative Savings YTD tracker
- Payment mode breakdown (UPI, Credit Card, Cash)
- Quarterly QoQ comparison
- Investment Rate % discipline tracker
- Interactive slicers for Month, Quarter, Category

## Key Insights
- Festive months (Oct + Dec) showed 40% higher spending
- Savings rate improved after July salary increment
- Investment rate grew from 18% to 26% by December
